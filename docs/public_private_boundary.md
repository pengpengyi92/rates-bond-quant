# Public / Private Boundary

This project is designed to be safe for a public GitHub portfolio.

## Can Publish

- synthetic macro / rates events
- mock yield curve data
- simplified public-domain-style schemas
- general rates research workflow
- generic scenario labels
- generic risk checklist
- software architecture
- tests and demo reports

## Do Not Publish

- employer internal materials
- bank client information
- internal credit / treasury / transaction records
- proprietary market data
- non-public trading views
- real portfolio positions
- internal approval processes
- screenshots from private systems
- identifiable colleague / client / desk information

## Safe Project Claim

```text
This is a public-safe research engineering demo for FICC rates-bond reasoning.
It uses synthetic data and simplified rules to demonstrate the workflow from
events and yield-curve state to scenario classification, hypothesis generation,
risk checklist, and human-reviewable report.
```

## Unsafe Claims To Avoid

Do not claim:

```text
production trading system
bank internal rates research engine
real portfolio optimizer
real trading signal
verified PnL
live rates strategy
```

## Resume Boundary

Good wording:

```text
Designed a public-safe FICC rates bond research OS that maps synthetic
macro/rates events and yield-curve snapshots into scenario classifications,
candidate hypotheses, risk checklists, and reviewable markdown reports.
```

Bad wording:

```text
Built a rates trading system for bank portfolio decisions.
```
